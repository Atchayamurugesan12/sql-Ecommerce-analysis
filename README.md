# Ecommerce SQL Data Analysis

## Project Overview
This project analyzes an ecommerce dataset using SQL to answer
business questions related to sales, customers, products, and revenue.

## Dataset
The dataset used in this project is sourced from Kaggle.

It contains ecommerce transaction data including:
- User ID
- Product ID
- Category
- Purchase Date
- Payment Method
- Price
- Discount
- Final Price

## Dataset Source:
Kaggle – Ecommerce Dataset  

## Tools Used
- SQL (MySQL)
- GitHub for project documentation
